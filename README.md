FORT/
-
  FortAtacadista.py is a program to webscrape FortAtacadista website and get the price of all its products. 
  its to be run daily on a scheduled task.
  
  FortSTD is a pipeline to take data appended by FortAtacadista.py and produce insights on Inflation.
  The main objective of this project is to keep track of market inflation, since gov stats in brazil cant be trusted.
  
FUTEBOL PORTUGUES/
-
GetFutData.py is a program to scrape Transfermarkt for every portuguese team stats over differente pages.
GetGameResults.py is a program to scrape flashscore for all rounds already played.
FutNovaRodada.py scrapes for the next round of games yet to be played.
FUTML.py takes the data from the 3 tables generated, joins them based of the next fixture and uses 2 ML methods to predict the outcome.
It uses Neural Network and Random Forest to predict the games results
